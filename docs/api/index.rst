=============
API reference
=============

.. warning:: Not all APIs will work correctly when called from Python if they
    require running in a UWP app or in an app with a package identity. Learn
    more and see an list of affected APIs at
    https://learn.microsoft.com/en-us/windows/apps/desktop/modernize/desktop-to-uwp-supported-api.

    It is possible to run Python with a package identity. However, this topic
    has not been fully explored yet.
